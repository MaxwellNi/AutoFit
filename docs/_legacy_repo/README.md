# Legacy Archive

This directory stores historical documentation that is no longer current for Phase 9 / V739 execution.

## Why These Files Were Moved Here

1. They describe superseded benchmarks, especially Phase 7/8 and the old V72/V73 development line.
2. They contain conclusions tied to deprecated or invalid AutoFit versions.
3. Leaving them in the top-level `docs/` surface caused future readers and tools to over-trust stale narratives.

## What This Archive Is Still Good For

1. Failure analysis history
2. Design ideas that may still inspire future versions
3. A record of what was tried, what failed, and why

## What This Archive Must NOT Be Used For

1. Current benchmark status
2. Current queue status
3. Current AutoFit baseline definition
4. Any paper-ready Phase 9 / V739 fact claim

## Current Truth Lives Here Instead

1. `docs/CURRENT_SOURCE_OF_TRUTH.md`
2. `docs/BLOCK3_MODEL_STATUS.md`
3. `docs/BLOCK3_RESULTS.md`
4. `docs/benchmarks/phase9_current_snapshot.md`
5. Root `AGENTS.md`
6. `.local_mandatory_preexec.md`
